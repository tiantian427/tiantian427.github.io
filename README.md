## Preview locally

Open `index.html` in a browser.

## Deploy on GitHub Pages

1. Create a GitHub repository. Recommended repository name:
   `tongpu-zhao.github.io`

2. Upload everything in this folder:
   - `index.html`
   - `styles.css`
   - the whole `assets` folder

3. Open the repository on GitHub.

4. Go to:
   `Settings` → `Pages`

5. Under `Build and deployment`, choose:
   `Deploy from a branch`

6. Select:
   - Branch: `main`
   - Folder: `/root`

7. Save. GitHub will generate a public website URL after a short wait.

## Deploy on Vercel

1. Push this folder to a GitHub repository.
2. Go to Vercel.
3. Import the repository.
4. Deploy.
